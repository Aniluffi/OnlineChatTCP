# OnlineChatTCP

Простой учебный чат на TCP-сокетах на C#/.NET 8.

Проект состоит из двух консольных приложений:
- **DOCS2** — TCP-сервер, который принимает подключения и рассылает сообщения другим клиентам.
- **Client2** — TCP-клиент для отправки и получения сообщений.

## Стек
- C#
- .NET 8
- `System.Net.Sockets`

## Структура репозитория
```text
OnlineChatTCP/
├─ DOCS2/                # solution и проект сервера
│  └─ DOCS2/
│     ├─ Program.cs      # точка входа сервера
│     └─ Server.cs       # логика TCP-сервера и рассылки
├─ Client2/              # solution и проект клиента
│  └─ Client2/
│     └─ Program.cs      # логика TCP-клиента
└─ README.md
```

## Как это работает
1. Сервер поднимается на `0.0.0.0:808` и ожидает подключения.
2. При подключении клиента сервер добавляет его в список активных клиентов.
3. Клиент отправляет строку текста на сервер.
4. Сервер получает сообщение и рассылает его всем остальным подключенным клиентам (кроме отправителя).

## Требования
- Установленный **.NET SDK 8.0+**

## Запуск
### 1) Запустить сервер
Из корня репозитория:
```bash
dotnet run --project DOCS2/DOCS2/DOCS2.csproj
```

Ожидаемый вывод:
```text
Сервер запущен. Ожидание подключений...
```

### 2) Запустить клиентов
Откройте два и более терминала и в каждом выполните:
```bash
dotnet run --project Client2/Client2/Client2.csproj
```

После запуска вводите сообщение в строке `Send:` и нажимайте Enter.

## Пример сценария
1. Запустить сервер.
2. Подключить Клиент A.
3. Подключить Клиент B.
4. В Клиенте A отправить сообщение `Привет!`.
5. В Клиенте B появится сообщение вида:
```text
Сообщение от 127.0.0.1:<порт>: Привет!
```
